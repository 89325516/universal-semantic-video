# Business Requirements Document

## English

### Goal

Create a public, interoperable foundation for Universal Semantic Video (USV): a semantic metadata layer above existing audio and video media.

### Context

Media localization still depends on platform-specific subtitles, burned-in text replacement, duplicated language masters, and opaque AI pipelines. USV addresses this by declaring semantic objects, localization intent, fallback behavior, rights, and provenance in a portable sidecar.

### Constraints

- Use existing media containers and standards where possible.
- Keep the first release limited to schema, examples, validator, and documents.
- Do not publish private source material or raw media.
- Repository-facing text is English, German, or both.

### Done When

The first release is accepted when `ACCEPTANCE.md` passes.

## Deutsch

### Ziel

Ein oeffentliches, interoperables Fundament fuer Universal Semantic Video (USV) schaffen: eine semantische Metadatenschicht oberhalb bestehender Audio- und Videomedien.

### Kontext

Medienlokalisierung haengt oft von plattformspezifischen Untertiteln, eingebrannter Textersetzung, mehrfachen Sprachmastern und undurchsichtigen KI-Pipelines ab. USV beschreibt deshalb semantische Objekte, Lokalisierungsabsicht, Fallback-Verhalten, Rechte und Herkunft in einem portablen Sidecar.

### Einschraenkungen

- Bestehende Mediencontainer und Standards werden bevorzugt.
- Der erste Release bleibt auf Schema, Beispiele, Validator und Dokumente begrenzt.
- Private Quellen und Rohmedien werden nicht veroeffentlicht.
- Repository-Inhalte sind auf Englisch, Deutsch oder beides beschraenkt.

### Fertig Wenn

Der erste Release ist fertig, wenn `ACCEPTANCE.md` erfuellt ist.
